##### 개발 범위

-	사용자가 회원가입 및 로그인을 할 수 있다.

-	튜터는 강좌를 생성할 수 있게 한다.

-	튜티는 희망하는 주제의 강좌를 찾는 구인 글을 만들 수 있게 한다.

-	댓글을 남겨 상호간 평가를 할 수 있게 한다.

##### 개발 포지션

-	Front-end: 김세정, 한종우

-	Back-end: 전판근, 신희주
